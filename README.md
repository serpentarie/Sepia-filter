# Sepia-filter

Реализованы фильтр сепия на языке C и на языке ассемблера.
Реализация на языке ассемблера использует векторные инструкции процессора SSE, за счет чего получено ускорение по сравнению с реализацией на C.

Написан тест, который сравнивает время работы двух реализаций.

[Для работы с изображениями использована библиотека работы с BMP файлами, реализована мною раннее.](https://github.com/serpentarie/Image-Transform)
